Présentation du Site SegPonsBois Laravel

Introduction

Ce site est développé en utilisant le framework Laravel, il permet aux utilisateur de passer des commande d'essences de bois, ou encore des produits fabriqué en bois. Ce site est un exemple de site e-commerce, il permet de gérer les utilisateurs, les produits, les commandes, etc. Il ne dispose pas de plateforme de paiement, car étant pour le collège Emiles-Combes ceux ci gère la facturation en interne. Ce site gère juste la prise de commande de la part des client en génerant un bon de commande que le client récupère, ainsi que les admin du site.

Fonctionnalités

  • Authentification et gestion des utilisateurs
  • CRUD (Create, Read, Update, Delete) pour les ressources
  • Gestion de commandes
  • Gestion des migrations et des bases de données

Technologies Utilisées

Le site utilise les technologies suivantes :

  • Laravel
  • PHP
  • MySQL
  • JavaScript
  • HTML & Tailwindcss
